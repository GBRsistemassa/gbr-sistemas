Como Usar o Desenvolvedor

Fluxo de criação de módulo:

1. Preencha o Nome (ID gerado automaticamente)

2. Escolha o ícone Font Awesome (ex: fas fa-users)

3. Se for submenu, informe o Menu Pai

4. Escreva o código HTML/JS na área de código

5. Clique em Preview para testar

6. Clique em Injetar Módulo para publicar

ASAAS INTEGRAÇÃO
Configure cobranças automáticas PIX/Boleto com dois modos de operação
⚡
Modo de Operação
Como o sistema reage ao receber pagamento confirmado pelo Asaas
DESATIVADO
🤖
Automático — Webhook Asaas RECOMENDADO
Quando Asaas confirmar pagamento PIX/Boleto, o webhook dispara e libera a licença automaticamente sem intervenção manual.
✅ Zero trabalho manual   ⚠️ Requer URL de webhook configurada no Asaas + Cloud Function
🧑‍💼
Manual — Confirmar no Console IDEAL PARA TESTES
Asaas cria as cobranças e notifica o admin, mas a liberação da licença é feita manualmente pelo console na aba Pendências PIX.
✅ Controle total   ✅ Ideal para fase de testes   🔧 Sem necessidade de webhook nem Cloud Function
Selecione um modo acima e depois ative

Configuração do Webhook
Como configurar no Asaas:
1. Acesse Asaas → Configurações → Integrações → Webhook
2. Cole a URL abaixo no campo "URL do Webhook"
3. Marque os eventos: PAYMENT_CONFIRMED, PAYMENT_RECEIVED, PAYMENT_OVERDUE
4. Defina um token de segurança e cole abaixo
⚠️ A URL de webhook requer um Firebase Cloud Function deployado no projeto. Enquanto estiver em testes, use o Modo Manual.
